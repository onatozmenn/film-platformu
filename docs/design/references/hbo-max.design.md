# HBO Max Design Reference

- Source: https://styles.refero.design/style/898f0127-d235-4832-bf33-ab21104f0529
- Assigned surface: Home and catalog
- Status: Distilled research reference; not an implementation contract

## Extracted Character

A sparse streaming catalog where black recedes and poster artwork appears illuminated. Typography and spacing carry hierarchy; surfaces remain flat, shadows and glass effects are absent, and a single accent marks interaction.

## Extracted System

| Concern | Refero observation |
|---|---|
| Canvas | `#000000`, secondary `#050409` |
| Text | `#ffffff`, muted `#b8b6bb`, tertiary `#89868e` |
| Source accent | `#3d6a99`, secondary `#71b8f2` |
| Typeface | Max Sans, weights 100-700 |
| Base spacing | 4px |
| Content width | 1280px |
| Section gap | 64px |
| Element gap | 16px |
| Source radius | 4.5-8px plus pills |

## Accepted Patterns

- Posters and cinematic stills lead; application chrome stays visually secondary.
- Full-width dark navigation and hero surfaces transition directly into content.
- Catalog sections are separated by rhythm rather than boxed page sections.
- Flat surfaces, hairline edges, and no cast shadows.
- Sparse, concise metadata around artwork.
- One interaction accent across the experience.

## Rejected Or Corrected

- Signal Blue and Glacier Beam do not enter the Film Platform palette.
- Max Sans is not used; typography remains DM Sans plus Source Serif 4.
- Pill controls, 8px button language, positive tracking, pricing-card polarity, and commerce styling are not borrowed.
- Pure white is reserved for short high-contrast labels rather than long body copy.
- No source poster, logo, copy, or exact layout is reproduced.

## Film Platform Translation

Home and catalog use a poster-first hierarchy: one photographic featured-film hero, immediately followed by dense film rails or grids on the uninterrupted canvas. Filters, navigation, and account controls are quiet enough that artwork remains the first visual read.
