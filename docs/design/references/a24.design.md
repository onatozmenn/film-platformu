# A24 Design Reference

- Source: https://styles.refero.design/style/6afa22a6-bec8-47c3-b5ee-5d11d64902cb
- Assigned surface: Film detail and editorial hierarchy
- Status: Distilled research reference; not an implementation contract

## Extracted Character

A film-title sequence translated into a website: monochrome bands, extreme typographic confidence, generous negative space, hairline structure, and film titles treated as visual artifacts instead of small database labels.

## Extracted System

| Concern | Refero observation |
|---|---|
| Palette | `#000000`, `#888888`, `#cacaca`, `#eeeeee`, `#ffffff` |
| Primary type | NB International Web, weights 400/500 |
| Metadata accent | NB International Mono Web |
| Display | Up to 74px with compressed line height |
| Section gap | 96px |
| Element gap | 9px |
| Source radius | 0px |
| Elevation | None |

## Accepted Patterns

- Film title as the dominant element of the detail page.
- Left-aligned title, year, runtime, classification, genres, and credits arranged as an editorial composition.
- Negative space around the title and synopsis rather than surrounding each field with a card.
- Hairline structural separators used only when spacing cannot communicate grouping.
- Restrained weight hierarchy and metadata that reads like a film programme.
- Full-bleed image/title bands where content warrants them.

## Rejected Or Corrected

- The global experience does not alternate into white page bands.
- Zero-radius everywhere, proprietary typography, negative tracking, and a third mono font are not adopted.
- Display type is capped by Film Platform's responsive scale rather than copied at 74px.
- The Acid Green action/rating accent remains available.
- Source titles, imagery, catalogue, and exact layouts are not reproduced.

## Film Platform Translation

Film detail becomes editorial without changing tokens: the backdrop creates the cinematic field, the title block sits left-aligned with concise metadata, the synopsis uses Source Serif 4, and credits form an unframed ordered list. Actions remain compact product controls rather than becoming promotional typography.
