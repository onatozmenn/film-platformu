# Open Film Artwork

These local catalog images represent explicitly licensed Blender open movies. Runtime pages do not hotlink the remote originals. File integrity is enforced by `src/content/open-film-catalog.json`.

| Local file | Official source | License evidence | Local transformation |
|---|---|---|---|
| `sintel.jpg` | Durian gallery, [`01a_comp_000197.jpg`](https://durian.blender.org/wp-content/uploads/2010/06/01a_comp_000197.jpg) | [CC BY 3.0 sharing statement](https://durian.blender.org/sharing/) | Original 2048x872 frame; pixels unmodified. |
| `tears-of-steel.jpg` | Mango gallery, [`02_dome.jpg`](https://mango.blender.org/wp-content/gallery/4k-renders/02_dome.jpg) | [CC BY 3.0 sharing statement](https://mango.blender.org/sharing/) | Original 1920x803 frame; pixels unmodified. |
| `elephants-dream.jpg` | Official Blender-hosted [`elephantsdream-720-h264-st-aac.mov`](https://download.blender.org/ED/elephantsdream-720-h264-st-aac.mov) | [CC BY 2.5 license statement](https://orange.blender.org/blog/creative-commons-license-2/) | One 720x405 frame extracted at 00:04:00 with FFmpeg; no other modification. |

Excluded project logos, trademarks, DVD covers, disc artwork, sponsor marks, magazine covers, and separately licensed soundtrack files are not used. Full films are ingested without editing and retain their original end-credit rolls. Visible film-detail attribution is generated from the reviewed manifest rather than duplicated UI literals.
